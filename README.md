# OmegaGoPlus
## 介绍
本科三年级AI课程大作业的进一步完善。

## 软件架构
1.  单机无AI版本
2.  单机有AI版本（三种AI）
3.  联机双人版本
4.  联机三人版本

**NOTE**: 联机版本很久未经测试

## 安装教程
1.  (Install One) pip ... -i https://pypi.douban.com/simple
2.  (Install All) pip install -r requirements.txt -i https://pypi.douban.com/simple
3.  (Save Env)    pip freeze > requirements.txt
